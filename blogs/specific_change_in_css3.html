Specific changes in CSS-3

Last updated 2 Days ago, 19 min

What is CSS?

CSS is the abbervation of Cascading Style Sheet which is a simple mechanism for adding styles in a given and linked HTML file. For example: fonts, colors, backgrounds e.t.c.

CSS3 is the latest evolution of CSS which is the new version of CSS2.

1. CSS3 Selectors

In addition to CSS2, CSS3 they allow for substring selection. We can use some REGEX for matching any element.
For example: using "*" for selecting an element with the requirement is satisfied anywhere in the element.


Pseudo-classes: a few more pseudo-classes were added. Some are:-

  • :root - to point the root element of the document.
  • :last-child - to select the last element of the parent element
  • :none - can be coupled with almost any other selector and used to get a element with negative style.

2. CSS3 Colors

The most notable change with these new color types is the ability to declare semitransparent colors.

  • RGBA: the fourth value or the alpha declares the opacity value which ranges between 1 and 0
  • Opacity: it sets the opaqueness of the element.

CSS3 provides the ability to add drop shadows to elements using the "box-shadow" property. This property lets you to specify the color, height, width, blur ...


Text shadow adds shadodws to individual characters in text nodes using the "text-shadow" property.
